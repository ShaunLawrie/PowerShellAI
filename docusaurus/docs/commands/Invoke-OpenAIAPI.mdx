## SYNOPSIS

Invoke the OpenAI API

## SYNTAX

```powershell
Invoke-OpenAIAPI [-Uri] <Object> [[-Method] <Object>] [[-Body] <Object>] [-NoProgress] [<CommonParameters>]
```

## DESCRIPTION

Invoke the OpenAI API

## EXAMPLES

### EXAMPLE 1

```powershell
Invoke-OpenAIAPI -Uri "https://api.openai.com/v1/images/generations" -Method Post -Body $body
```
