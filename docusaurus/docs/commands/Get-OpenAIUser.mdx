## SYNOPSIS

Get OpenAI User Information

## SYNTAX

```powershell
Get-OpenAIUser [-OrganizationId] <String> [<CommonParameters>]
```

## DESCRIPTION

Returns an overview of the user's OpenAI organization information

## EXAMPLES

### EXAMPLE 1

