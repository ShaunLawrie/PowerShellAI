## SYNOPSIS

Sets the Azure OpenAI API endpoint, deployment name, API version, and API key.

## SYNTAX

```powershell
Set-AzureOpenAI [-Endpoint] <Object> [-DeploymentName] <Object> [-ApiVersion] <Object> [-ApiKey] <Object>
 [<CommonParameters>]
```

## DESCRIPTION

Sets up Azure OpenAI as the chat API provider.
Use `Set-ChatAPIProvider -Provider OpenAI` to point to the public OpenAI

## EXAMPLES

### EXAMPLE 1

```powershell
Set-AzureOpenAI `
    -Endpoint https://anEndpoint.openai.azure.com/ `
    -DeploymentName aName `
    -ApiVersion 2023-03-15-preview `
    -ApiKey aKey
```
