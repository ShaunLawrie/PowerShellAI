## SYNOPSIS

Enables chat persistence.

## SYNTAX

```powershell
Enable-ChatPersistence
```

## DESCRIPTION

The Enable-ChatPersistence function sets the $Script:ChatPersistence variable to $true, which enables chat persistence.
