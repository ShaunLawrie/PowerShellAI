## SYNOPSIS

Get a summary of OpenAI API usage

## SYNTAX

```powershell
Get-OpenAIUsage [[-StartDate] <DateTime>] [[-EndDate] <DateTime>] [-OnlyLineItems] [<CommonParameters>]
```

## DESCRIPTION

Returns a summary of OpenAI API usage for your organization.
All dates and times are UTC-based, and data may be delayed up to 5 minutes.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-OpenAIUsage -StartDate '2023-03-01' -EndDate '2023-03-31'
```
